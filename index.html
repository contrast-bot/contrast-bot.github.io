Meet Contrast

A versatile Discord bot built with TypeScript, SQLite and Discord.js

Add to Discord View on GitHub 
const contrast = new Client({
  intents: [
    GatewayIntentBits.Guilds,
    GatewayIntentBits.GuildMessages,
    GatewayIntentBits.MessageContent,
    GatewayIntentBits.GuildMembers
  ]
});

await contrast.login(config.token);
// Contrast Bot is now online!

Features

Fun Commands

Entertaining commands like Rock Paper Scissors, jokes, quotes, slots, russian roulette and more.

Economy

Personal balance, daily rewards, coins transfer, GAMBLING and more!

Social Features

Reputation system, quotes, and other social interaction features.

Leveling System

Powerful global leveling system with XP and leaderboards.

SQLite Database

Persistent storage for user data, quotes, reputation, and more.

TypeScript

Built with TypeScript for type safety and better code organization.

Popular Commands

/ ping

Check the bot's latency and API response time

/ help

Display all available commands and their usage

/ whois [user]

Display detailed information about a user

/ server-info

Display information about the current server

/ rps [choice]

Play Rock Paper Scissors against the bot

/ remind [time] [message]

Set a reminder for yourself

/ quote [add/random/get]

Manage and display quotes

/ rep [give/take/check]

Manage user reputation
